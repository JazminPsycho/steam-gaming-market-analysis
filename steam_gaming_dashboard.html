// INSIGHT 01
RPG: alta demanda, menor saturación
Con 127.919 propietarios promedio y solo 17.095 juegos compitiendo, el RPG tiene el mejor ratio demanda/competencia del mercado. Un RPG narrativo con identidad latinoamericana clara tiene chances reales de destacar.
// INSIGHT 02
Action engaña: mucha demanda, más competencia
Action parece atractivo por sus 124.452 owners promedio, pero compite con 39.142 juegos. Para un estudio sin presupuesto de marketing, entrar a Action sin diferencial claro es un error estratégico.
// INSIGHT 03
Indie como tag no es una estrategia
El género Indie tiene el peor índice de oportunidad: 0.97. Con 52.781 juegos es el más saturado de todos. Ser indie es una descripción del estudio, no del juego. El género del juego importa más.
// RECOMENDACIÓN FINAL
Si sos un estudio indie con recursos limitados, los datos hablan claro.
No compitas en Action ni uses Indie como tu género principal. Los datos de 185.397 juegos de Steam muestran que RPG y Strategy tienen la mejor relación entre demanda y competencia. Un RPG narrativo o un juego de estrategia con identidad latinoamericana clara tiene más probabilidades de destacar que el shooter número 39.000 de Steam. La clave no es elegir el género más popular, sino el que mejor equilibra demanda alta con competencia manejable.
